# nesbox

> Nesbox does not provide binaries, nesbox provides source code with the intention of it being modified to suit a game or developers specific needs.
> 
> This means you are likely to execute a modified version of nesbox, please ensure you trust whatever applications you run.

Nesbox is a WIP emulator using `SDL3` supporting `net9` with `AOT` in `C#13` for use with Windows, Linux and MacOS. This tool is not for playing commercial games or beginner use. Users are required to have capability with C sharp and an understanding of the hardware of the NES. It's designed with the intention of providing a framework for accurate and fast emulation of WIP/experimental hardware with reduced attention on quality of use and enhancement of experience in comparison to native hardware experience. To find user docs, please visit the docsite at [docsite](https://nesbox.readthedocs.io/en/latest/). It is recommended to use at least a 1080p60 monitor.
